
# Tic Tac Toe Game

Interactive browser-based Tic Tac Toe game with selectable board size and optional computer opponent.

## About the Project

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and basic game logic.

The game allows users to choose the board size (from 3x3 to 7x7), play in two-player mode or against a computer, and interact with a dynamic game board built directly in the DOM.

## Features

- Dynamic board size selection (3x3 to 7x7)
- Two-player mode
- Optional computer opponent
- Random computer moves
- Modal windows for game results
- Interactive UI with DOM manipulation

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## What I Practiced

- Working with two-dimensional arrays
- DOM manipulation
- Event handling (addEventListener)
- Game state management
- Conditional logic
- Loops (for, nested loops)
- Dynamic UI updates
- Basic computer move logic
